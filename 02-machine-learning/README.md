# Machine Learning

## What Is Machine Learning?

Machine Learning allows computers to learn patterns from data.

Instead of writing rules manually, models learn automatically.

## Types

### Supervised Learning

Uses labeled data.

Examples:

- Spam Detection
- House Price Prediction

### Unsupervised Learning

Uses unlabeled data.

Examples:

- Customer Segmentation
- Clustering

### Reinforcement Learning

Learns through rewards.

Examples:

- Robotics
- Gaming

## Common Algorithms

### Linear Regression

Predicts continuous values.

Example:

House Price Prediction.

### Logistic Regression

Binary classification.

Example:

Spam or Not Spam.

### Decision Tree

Rule-based learning.

### Random Forest

Collection of Decision Trees.

### KNN

Finds nearest neighbors.

## Model Lifecycle

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Training
5. Evaluation
6. Deployment
