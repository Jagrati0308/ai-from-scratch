# Computer Vision

## What Is Computer Vision?

Computer Vision enables machines to understand images and videos.

## Image Representation

Images are matrices of pixels.

RGB Image:

Height × Width × 3

## Common Tasks

### Classification

Identify image category.

### Object Detection

Locate objects.

### Segmentation

Pixel-level understanding.

### OCR

Extract text from images.

## CNN

Convolutional Neural Networks are specialized for images.

Components:

- Convolution Layer
- Pooling Layer
- Fully Connected Layer

## Applications

- Face Recognition
- Medical Imaging
- Autonomous Vehicles
- Logo Detection
