# Transformers

## Why Transformers?

Transformers solve limitations of RNNs and LSTMs.

## Attention Mechanism

Allows models to focus on important words.

## Self Attention

Each word attends to all other words.

## Multi Head Attention

Multiple attention mechanisms run in parallel.

## Positional Encoding

Provides sequence order information.

## Encoder

Processes input.

## Decoder

Generates output.

## Applications

- GPT
- BERT
- T5
- Llama
