# Retrieval Augmented Generation

## Problem

LLMs forget information and may hallucinate.

## Solution

RAG retrieves external knowledge before generating responses.

## Pipeline

1. Documents
2. Chunking
3. Embeddings
4. Vector Database
5. Retrieval
6. Generation

## Vector Databases

- FAISS
- Chroma
- Pinecone

## Embedding Models

- BGE
- E5
- OpenAI Embeddings
